# Automated Email Sending Service

This repository contains an Automated Email Sending Service implemented using Java Mail API, Dependency Injection, Mockito for testing, Parameterized Tests, and Unit Tests.

## Table of Contents

- [Introduction](#introduction)
- [Features](#features)
- [Prerequisites](#prerequisites)


## Introduction

The Automated Email Sending Service is a Java-based application that allows you to send emails automatically using the Java Mail API. It is designed with modularity in mind, utilizing dependency injection for easy integration and testing. The service also includes comprehensive unit tests with parameterized testing techniques to ensure robustness and reliability.

## Features

- Send automated emails using Java Mail API.
- Flexible configuration options for email settings.
- Dependency Injection for easy integration and extension.
- Unit tests to validate the functionality of the service.
- Parameterized tests for testing multiple scenarios efficiently.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Java Development Kit (JDK) 8 or higher installed.
- Gradle build tool installed (or use the provided wrapper).
